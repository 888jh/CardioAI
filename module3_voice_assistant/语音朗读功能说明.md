## 🔧 技术实现

### 1. TTS 引擎

**库**: `pyttsx3`

**特点**:
- 跨平台（Windows/Mac/Linux）
- 使用系统内置语音引擎
- 无需网络连接
- 支持中文

### 2. 实现方式

**独立进程执行**（避免 Flask 冲突）:
```python
# 在独立的 Python 进程中运行 TTS
subprocess.run([python_exe, tts_script])
```

**原因**:
- pyttsx3 的 COM 组件与 Flask 调试模式冲突
- 独立进程避免文件监控问题
- 提高稳定性

### 3. 音频参数

```python
rate = 150      # 语速（每分钟字数）
volume = 0.9    # 音量（90%）
format = WAV    # 输出格式
```

## 🐛 故障排除

### 问题 1: 听不到声音

**原因**: 音量设置、浏览器静音

**解决**:
1. 检查系统音量
2. 检查浏览器音频权限
3. 手动点击播放按钮

### 问题 2: 响应时间过长

**原因**: TTS 引擎初始化、文本过长

**解决**:
1. 首次调用会慢，之后正常
2. 缩短回答长度（在 System Prompt 中设置）

### 问题 3: 语音不是中文

**原因**: 系统未安装中文语音

**解决**:
1. Windows 设置 → 语音 → 下载中文语音包
2. 或使用英文语音（会尝试朗读中文）

### 问题 4: TTS 失败，听到音调

**原因**: pyttsx3 初始化失败，使用备用方案

**解决**:
1. 检查 pyttsx3 是否正确安装
2. 重启应用
3. 查看服务器日志排查错误
